Extjs中textfield的悬浮提示

2017-02-22

在Extjs中有时候我们需要textfield的提示信息，但是我们发现textfield并没有这样的配置项。这时候我们就要另想方法：我们需要在鼠标悬停在textfield组件的时候进行信息的提示，我们就需要在textfield中来监听，
这里有两种进行textfield悬停提示的方法：

  1. 进行固定信息的提示：需要在监听中用render方法
    render: function (field, p) {
       Ext.QuickTips.init();
       Ext.QuickTips.register({
             target: field.el,
             text: '这是textfield的固定信息提示！'
       })
    }

  2. 进行文本信息提示：提示信息为textfield的文本信息
