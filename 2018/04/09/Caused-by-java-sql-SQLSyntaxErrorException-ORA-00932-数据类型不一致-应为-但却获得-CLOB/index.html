总是报：ORA-00932: 数据类型不一致: 应为 -, 但却获得 CLOB


是由于这个t0.INQUIRYREQUIRE字段clob字段。

第一种解决方法： t0.INQUIRYREQUIRE 改成 to_char(t0.INQUIRYREQUIRE)。

第二种解决方法：去掉distinct 去重。
